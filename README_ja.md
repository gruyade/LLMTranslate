# LLMTranslate (日本語)

[English Version (README.md)](README.md)

Vision機能を備えたLLM（OpenAI互換API）を使用して、Windowsデスクトップ画面を翻訳するツール。

## ✨ 主な機能

- **画面オーバーレイ**: 翻訳範囲を指定するリサイズ・移動可能な枠を表示。（枠の色は設定から変更可能。）
- **LLMによる翻訳**: キャプチャした画像をOpenAI互換API（LM Studio, OpenAI等）に送信して翻訳。
- **2種類の結果表示モード**:
  - **別ウィンドウ（バブル）**: 翻訳結果を枠のすぐ横に半透明フローティングウィンドウで表示。
  - **枠内オーバーレイ**: 翻訳結果をキャプチャ枠の内側に半透明で直接重ねて表示。
- **枠上ボタン**: オーバーレイ枠の右側に4つのボタン（⚙ 設定、▶ 翻訳、A/M モード、▣/□ 表示モード）を配置し、素早く操作可能。
- **手動翻訳モード**: 枠上のボタンまたはタスクトレイメニューから翻訳を実行。
- **自動監視モード**: 枠内の変化を自動検出し、テキストが変わった時のみ自動で翻訳。
- **多言語UI**: 10言語に対応（日本語、英語、フランス語、ドイツ語、タイ語、中国語、ポルトガル語、スペイン語、韓国語）。
- **タスクトレイ常駐**: 全ての操作にタスクバーのアイコンからアクセス可能。
- **詳細な推論設定**: Temperature, Max Tokens, Top Pなど、LM Studio準拠のパラメータを調整可能。
- **プリセット管理**: APIサーバーやモデルごとに設定を保存し、素早く切り替え。

## 🎯 想定利用シーン

### 作者の想定
- 自分が読める言語に対応していない、または翻訳品質が低いゲームのテキスト翻訳
- コピペできないテキストの翻訳手間の削減

### AIのおすすめ活用例
- 未翻訳の海外ソフトウェアUIの操作補助
- 外国語配信のチャット欄の監視翻訳（自動監視モードで新着メッセージを自動翻訳）
- リモートデスクトップや仮想マシン上の外国語テキストの翻訳
- コピー保護されたPDFやDRM付き電子書籍の翻訳
- ビデオ会議で共有された外国語画面のリアルタイム翻訳（スライドが切り替わるたびに自動翻訳）
- Canvas / WebGLベースのWebアプリなど、ブラウザ組み込み翻訳が効かない画面の翻訳
- 外国語学習時の参考訳確認


   ```bash
   pip install -r requirements.txt
   ```

3. 起動:
   ```bash
   python run.py
   ```

## 📖 使い方

### 基本操作

1. アプリを起動するとタスクトレイにアイコンが表示されます。
2. 枠線が表示されます。上辺のグラブハンドルをドラッグして移動、四隅のハンドルでリサイズします。（枠の色は設定から変更可能。）
3. 翻訳したいテキストが枠内に入るように合わせます。
4. オーバーレイ枠右側の **▶ 翻訳ボタン** をクリックするか、トレイアイコン右クリックから「翻訳を実行」を選択します。
5. 選択した表示モード（別ウィンドウまたは枠内オーバーレイ）で翻訳結果が表示されます。

### 操作方法

オーバーレイ枠の右側に4つのボタンが表示されます：

| ボタン | アイコン | 説明 |
|--------|----------|------|
| 設定 | ⚙ | 設定ダイアログを開く |
| 翻訳 | ▶ / ■ | 翻訳を実行（翻訳中は ■ 表示） |
| モード | A / M | 自動監視（A）/ 手動（M）モードを切り替え |
| 表示モード | ▣ / □ | 表示モードを切り替え（▣ = 枠内オーバーレイへ切替、□ = 別ウィンドウへ切替） |

- **トレイメニュー**: 設定、プリセット、手動操作にアクセスできます。

### ショートカットキーについて

グローバルショートカット（`Ctrl+Shift+T` など）も設定されていますが、アクティブなウィンドウの状態によっては動作しない場合があります。確実な操作にはUI上のボタンを使用することを推奨します。

### 自動監視モード

有効にすると、一定間隔で枠内をスキャンし、大きな変化（テキストの更新など）を検出したときのみ自動的に翻訳リクエストを送信します。

## ⚙️ 設定

タスクトレイ右クリック -> 「設定...」から設定ダイアログを開きます。

### サーバー設定

- **API Base URL**: OpenAI互換エンドポイント（例: LM Studioは `http://localhost:1234/v1`）。入力欄の下に実際のエンドポイントURLがリアルタイムでプレビュー表示されます。
- **API Key**: APIキー（LM Studioの場合は空でOK）
- **Model**: モデル名（例: `gpt-4o`, `lmstudio-community/...`）

### 推論パラメータ

LM Studioの設定項目に準拠。生成のランダム性や最大トークン数などを調整できます。

### 表示設定

- **翻訳結果の表示方法**: 2つのモードから選択できます：
  - **別ウィンドウ（バブル）**: 翻訳結果を枠の横に半透明フローティングウィンドウで表示。
  - **枠内オーバーレイ**: 翻訳結果をキャプチャ枠の内側に直接重ねて表示。
- **枠内表示設定**（枠内オーバーレイ選択時に有効）:
  - **不透明度**: 枠内オーバーレイパネルの透明度。
  - **最大高さ比率**: 枠の高さに対するオーバーレイパネルの最大高さの割合（10〜80%）。

### プリセット

設定画面上部のバーから、現在の設定を保存・読み込み・削除できます。OpenAI用、ローカルLLM用などで使い分けると便利です。

## 🔗 LM Studio との連携

1. LM Studioを起動し、**Vision対応モデル**をロードします。
2. Local Serverを起動します（デフォルト: `http://localhost:1234`）。
3. LLMTranslateの設定を以下のように変更します:
   - **API Base URL**: `http://localhost:1234/v1`
   - **API Key**: (空欄)
   - **Model**: LM Studioに表示されているモデル識別子
